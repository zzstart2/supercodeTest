Drop-in Integration

5 分钟接入，
像用 OpenAI 一样用 Vela-alpha

替换一行 base_url，就能接入统一的 API 网关、透明计量与可观测性。Python、Node、cURL 官方支持。

阅读文档 → 在线试用 
from openai import OpenAI

client = OpenAI(
    base_url="https://api.vela.dev/v1",
    api_key="va-xxxxxxxxxxxx",
)

response = client.chat.completions.create(
    model="vela-chat-pro",
    messages=[
        {"role": "user",
         "content": "用 Python 实现快速排序"}
    ],
    stream=True,
)

for chunk in response:
    print(chunk.choices[0].delta.content, end="")
